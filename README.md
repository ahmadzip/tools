## Getting Started

1. Clone the repo
   ```sh
   git clone https://github.com/ahmadzip/tools.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the app
   ```sh
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
